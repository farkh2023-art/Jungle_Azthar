Configuration Détaillée

1. Configuration Claude API

Pour intégrer Claude API d'Anthropic :

const claudeAPI = { apiKey: 'sk-ant-api03-...', baseURL: 'https://api.anthropic.com', model: 'claude-3-sonnet-20240229', maxTokens: 4096 }; // Exemple d'appel const response = await fetch('https://api.anthropic.com/v1/messages', { method: 'POST', headers: { 'Authorization': `Bearer ${claudeAPI.apiKey}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ model: claudeAPI.model, max_tokens: claudeAPI.maxTokens, messages: [{ role: 'user', content: 'Hello!' }] }) });
Gardez votre clé API secrète et ne l'exposez jamais côté client.

2. Configuration DeepSeek API

Pour intégrer DeepSeek pour les tâches de code :

const deepSeekAPI = { apiKey: 'sk-deepseek-...', baseURL: 'https://api.deepseek.com', model: 'deepseek-coder', temperature: 0.2 }; // Exemple pour génération de code const codeResponse = await fetch('https://api.deepseek.com/v1/chat/completions', { method: 'POST', headers: { 'Authorization': `Bearer ${deepSeekAPI.apiKey}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ model: deepSeekAPI.model, messages: [{ role: 'user', content: 'Generate a Python function to sort a list' }], temperature: deepSeekAPI.temperature }) });

3. Configuration ElevenLabs TTS

Pour intégrer ElevenLabs comme TTS premium :

const elevenLabsAPI = { apiKey: 'sk_elevenlabs_...', baseURL: 'https://api.elevenlabs.io/v1', voiceId: 'voice_id_here', model: 'eleven_multilingual_v2' }; // Générer audio const audioResponse = await fetch(`${elevenLabsAPI.baseURL}/text-to-speech/${elevenLabsAPI.voiceId}`, { method: 'POST', headers: { 'Accept': 'audio/mpeg', 'xi-api-key': elevenLabsAPI.apiKey, 'Content-Type': 'application/json' }, body: JSON.stringify({ text: 'Bonjour, ceci est un test', model_id: elevenLabsAPI.model, voice_settings: { stability: 0.5, similarity_boost: 0.5 } }) });

4. Configuration Azure Speech (Fallback)

Pour configurer Azure Speech comme solution de secours :

const azureSpeechAPI = { subscriptionKey: 'your_subscription_key', region: 'westeurope', endpoint: 'https://westeurope.tts.speech.microsoft.com/cognitiveservices/v1' }; // Configuration fallback const azureTTS = async (text) => { const response = await fetch(azureSpeechAPI.endpoint, { method: 'POST', headers: { 'Ocp-Apim-Subscription-Key': azureSpeechAPI.subscriptionKey, 'Content-Type': 'application/ssml+xml', 'X-Microsoft-OutputFormat': 'audio-16khz-128kbitrate-mono-mp3' }, body: `<speak version='1.0' xml:lang='fr-FR'> <voice xml:lang='fr-FR' xml:gender='Female' name='fr-FR-DeniseNeural'> ${text} </voice> </speak>` }); return response.blob(); };

5. Configuration OpenAI TTS (Standard)

Pour configurer OpenAI TTS comme solution standard :

const openAITTS = { apiKey: 'sk-openai-...', baseURL: 'https://api.openai.com/v1', model: 'tts-1', voice: 'alloy' }; // Générer audio avec OpenAI const generateSpeech = async (text) => { const response = await fetch(`${openAITTS.baseURL}/audio/speech`, { method: 'POST', headers: { 'Authorization': `Bearer ${openAITTS.apiKey}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ model: openAITTS.model, input: text, voice: openAITTS.voice, response_format: 'mp3' }) }); return response.blob(); };
